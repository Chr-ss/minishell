 This project is about creating a Bourne-\/\+Again Shell (BASH) replica for the minishell project at Codam. Minishell is therefore a user-\/mode program that implements a Command Line Interface (CLI).

In general terms what this program does is\+:


\begin{DoxyItemize}
\item Print a prompt
\item Read a line
\item Parse
\item Execute
\item Repeat
\end{DoxyItemize}

{\itshape key bindings}
\begin{DoxyItemize}
\item arrow up / down for history.
\item ctrl-\/C displays a new prompt on a new line.
\item ctrl-\/D exits the shell.
\item ctrl-\/\textbackslash{} does nothing.
\end{DoxyItemize}

{\bfseries{Built-\/ins}}


\begin{DoxyItemize}
\item echo with option -\/n
\item cd with only a relative or absolute path
\item pwd with no options
\item export with no options
\item unset with no options
\item env with no options or arguments
\item exit with no options
\end{DoxyItemize}
